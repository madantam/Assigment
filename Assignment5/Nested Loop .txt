Nested Loop :
-----------------
A loop can be nested inside another looop 

nested loop consisits of a outer loop and one or more inner loop .

each time the outer loop is excuted then inner loop will executed again

example :
----------

for( int i =0 ; i<=3 ; i++)
{
  for(int j =0 ; j<2 ; j++)
  {
     System.out.println(i + " " + j);
  }
   System.out.println()

}

Output :
-----------
 0 0
 0 1

1 0
1 1

2 0 
2 1
